# 第一区服：RIME 雾凇

## 这是什么服务器？

RIME 雾凇第一区服是一个偏原版体验的 Minecraft 纯生存服务器，也支持生电玩法和长期工程建设。

- 名字：RIME 雾凇
- 类型：纯生存 / 生电
- Minecraft 版本：1.21.11
- 服务端：Leaves

## 新玩家推荐流程

1. 使用推荐版本进入服务器。
2. 进服后先打开菜单：`/menu`、`/rime` 或 `/help`。
3. 在主城阅读公告和提示。
4. 使用 `/rtp` 随机传送到野外。
5. 找到合适地点后使用 `/sethome home` 设置家。
6. 需要回主城时使用 `/spawn`。
7. 需要回家时使用 `/home home`。
