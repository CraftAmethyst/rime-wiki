# 常见问题

本页整理第一区服玩家常见的连接、登录、语音和客户端问题。处理前可以先重启一次客户端，很多临时问题会在重启后恢复。

## 网络与连接

| 问题 | 处理方式 |
| --- | --- |
| 入服后延迟特别高 | 可以尝试使用 [EasyTier 备用连接](./tips#easytier-备用连接)。 |
| 显示 `未知的主机` | 先检查服务器地址是否输入正确，再尝试刷新服务器列表。 |
| 显示 `Connection refused: getsockopt` | 可能是本地网络、DNS 或服务器维护导致。可以先重启客户端和电脑，再重新进入。 |
| 显示 `无法将你转发到...` | 可能是服务器维护或跨服转发异常。等待一会儿再试；仍未解决时，把完整报错发到群里。 |
| 添加服务器后一直显示检测中 | 可以安装 [Fast IP Ping](https://www.mcmod.cn/class/11895.html) 模组；如果已经有 TweakerMore，可以开启 `yeetServerIpReversedDnsLookup` 选项。 |

::: tip DNS 刷新
如果一直无法解析服务器地址，可以在 Windows 的命令提示符中运行 `ipconfig /flushdns`，然后重启电脑再试。
:::

## 登录与验证

| 问题 | 处理方式 |
| --- | --- |
| 显示 `认证服务器处于宕机状态中。请稍后再试，抱歉！` | 这是 Mojang / Microsoft 验证服务器暂时不可用，重启客户端或稍后再试。 |
| 显示 `无法验证用户名` | 先重启客户端并重新登录正版账号。仍然不行时，等待验证服务器恢复。 |
| 显示 `Couldn't verify username because servers are unavailable.` | 含义同上，通常不是客户端配置问题。 |
| 显示 `Authentication servers are down. Please try again later.` | 含义同上，稍后再试。 |

## 语音聊天

| 问题 | 处理方式 |
| --- | --- |
| 听不到别人说话 | 按 `Y` 打开语音聊天设置，重新选择输出设备。 |
| 别人听不到我的声音 | 按 `Y` 打开语音聊天设置，重新选择输入设备。 |
| 设备选项不对 | 重新插拔耳机或麦克风后，再打开语音聊天设置选择设备。 |

语音聊天设置底部有两个设备按钮：左边选择输入设备，右边选择输出设备。

## 客户端

| 问题 | 处理方式 |
| --- | --- |
| 启动器打不开 | 先确认已经安装 JDK 25，详见 [客户端安装](./client-install)。 |
| 首次启动很慢 | 首次启动会下载和安装客户端文件，等待完成即可。 |
| 不确定是否用对客户端 | 重新下载群文件中的第一区服客户端，并解压到空文件夹后启动。 |

::: info 仍未解决
如果以上方法都无效，请把完整报错截图、使用的客户端版本、网络运营商和你已经尝试过的方法发到群里，方便定位问题。
:::
