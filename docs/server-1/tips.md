# 小技巧与教程

这里整理第一区服常用的补充教程。遇到连接、语音或客户端问题时，也可以先看 [常见问题](./faq)。

## EasyTier 备用连接

如果你连接服务器时延迟很高、丢包明显，或者运营商线路绕路，可以尝试使用 EasyTier 备用连接。移动宽带玩家更容易遇到这种情况，其他网络环境如果很卡也可以尝试。

::: tip 小提示
EasyTier 已经配置好，正常情况下只需要下载、解压并运行启动脚本。启动后仍然使用原来的服务器地址进服。
:::

## 下载 EasyTier

在群文件中打开 `EasyTier` 文件夹。

![EasyTier 文件夹](/images/群文件-圈出easytier文件夹.png)

根据自己的系统下载对应版本。

![EasyTier 下载文件](/images/群文件-EasyTier.png)

| 系统 | 下载后运行 |
| --- | --- |
| Windows | `启动.bat` |
| macOS | `启动.sh` |
| Linux | `run.sh` |

## 使用方法

1. 把下载的压缩包解压到一个单独文件夹。
2. 运行对应系统的启动脚本。
3. 保持 EasyTier 窗口运行，不要关闭。
4. 打开 Minecraft，使用原来的服务器地址进入第一区服。

![EasyTier 启动脚本](/images/文件夹-圈出启动脚本.png)

## 注意事项

- 如果运行脚本后没有反应，先确认压缩包已经完整解压。
- Windows 提示安全警告时，确认文件来自群文件后再允许运行。
- 进服后如果延迟没有改善，可以关闭 EasyTier，恢复原来的直连方式。
- EasyTier 只是备用连接方式，不需要修改客户端或服务器地址。
