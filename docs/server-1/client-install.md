# 第一区服客户端安装

本页适用于第一次进入 RIME 雾凇第一区服的玩家。按顺序完成 JDK、客户端和启动器登录即可进入服务器。

## 准备工作

开始前请先确认：

- 已经加入服务器群，并且可以打开群文件。
- 电脑上有解压软件，例如 7-Zip、Bandizip 或 WinRAR。
- 已经准备好正版 Minecraft 账号。

::: tip 小提示
如果你的电脑已经安装过 JDK 25，可以直接跳过 JDK 安装，继续下载客户端。
:::

## 安装 JDK 25

在群文件中打开 `JDK环境` 文件夹。

![JDK 环境文件夹](/images/群文件-圈出JDK环境.png)

进入文件夹后，下载 `JDK25` 安装包。

![JDK25 安装包](/images/群文件-圈出JDK25.png)

下载完成后打开安装包，安装界面出现后点击 `Next`。

![JDK25 安装界面](/images/JDK25安装-圈出Next按钮.png)

后续保持默认选项，一直点击 `Next`，直到安装完成。

## 下载并解压客户端

回到群文件，打开客户端文件夹。

![客户端文件夹](/images/群文件-圈出客户端.png)

选择第一区服客户端并下载。

![第一区服客户端](/images/群文件-圈出区服客户端.png)

下载完成后，把客户端压缩包解压到一个空文件夹中。建议单独新建文件夹，不要直接解压到桌面或下载目录。

## 启动游戏

打开解压后的客户端文件夹，找到自带的 PCL 启动器。

![PCL 启动器](/images/文件夹-圈出启动器.png)

双击启动器后，先配置正版登录。首次启动时 PCL 会自动安装游戏文件和客户端环境，等待安装完成后再进入游戏。

进入游戏后，在多人游戏中选择服务器即可加入第一区服。

## 常见问题

| 问题 | 处理方式 |
| --- | --- |
| 启动器打不开 | 先确认 JDK 25 已安装完成，再重新打开启动器。 |
| 找不到启动器 | 确认客户端压缩包已经完整解压，不要在压缩包预览窗口里直接启动。 |
| 首次启动很慢 | 第一次启动会下载和安装客户端文件，等待完成即可。 |
| 登录后进不去服务器 | 确认使用的是第一区服客户端，并检查游戏里的报错提示。 |
